# BotForge Monorepo

## Structure
- `apps/bot` - Discord bot (TypeScript)
- `apps/dashboard` - Next.js dashboard (TypeScript)
- `packages/database` - Shared MongoDB client
- `packages/shared-types` - Shared TypeScript types
- `packages/config` - Shared config defaults

## Setup
```bash
pnpm install
pnpm dev
```
